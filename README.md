SafeBank is a **menu-driven banking application** written in **x86 Assembly (MASM)** using the **Irvine32 library**.  
It simulates basic banking operations such as creating an account, depositing money, withdrawing funds, printing account details, and resetting the account.

This project is designed for **educational purposes**, especially for students learning Assembly Language with MASM.

---

Features

 Create Account (Saving / Current)  
 Enter Account Holder Name and PIN  
 Initial Balance Setup  
 Welcome Message After Account Creation  
 Deposit Money  
 Withdraw Money (with insufficient balance check)  
 Print Account Details  
 Reset Account (with confirmation)  
 Menu-driven Interface  
 Clean Screen Transitions  
 Input Validation  

---

 Technologies Used

- **MASM (Microsoft Macro Assembler)**
- **Irvine32 Library**
- **x86 Assembly (32-bit)**
- Windows Console

*(Make sure Irvine32 files are properly configured in your MASM setup.)*
